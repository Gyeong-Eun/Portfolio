## 3. 분석 방법
- [`소스 코드 (최적화)`](https://github.com/Gyeong-Eun/portpolio/blob/master/Optimal_Location/code/폐의약품_set_covering&P_median&세대수.ipynb)

**Set-covering → 폐의약품 수거함의 최소 설치 개수 산출**
- 시설 간의 최대 허용거리를 모두 커버할 수 있는 최소의 입지 수를 결정하는 의사결정 기법인 set-covering 모형을 적용
- 모든 사람들에게 폐의약품 수거 서비스를 제공하면서 최소의 폐의약품 수거시설의 개수를 결정하기 위해 위에서 설정한 후보지와 블록화한 수요지 데이터를 바탕으로 set-covering 모형을 적용

**P-median → 폐의약품 수거함의 최적 입지 선정**
- p-median 모델은 시설 위치와 수요 지점 간의 평균 거리를 최소화하는 모델로, 수요자와 시설물 간의 총 이동거리를 최소화하여 형평성을 증대시키는 것이 목적
- 폐의약품 수거함은 공간적으로 편의성과 접근성이 중요하기 때문에 후보 지점과 수요 지점 간의 총 이동거리를 최소화하는 p-median 모델을 적용
- p개 이하의 폐의약품 수거함의 최적 입지를 새롭게 선정하였으며, 이때 p값은 set-covering을 통해 산출된 값을 사용

## 5. 분석 결과 및 인사이트 

**폐의약품 수거함 개수 증가**
- 유동 인구의 접근성을 고려해 대로변 인근 공공시설에 설치된 수거함이 산지 근처 수요지를 커버할 수 있도록 재배치한 결과로 판단하였다.


**폐의약품 수거함과 수요지까지의 거리**
- 기존 수거함이 커버하는 수요지까지의 거리 총합은 88,255m였으나, 최적 입지에서는 58,405m로 총 29,850m 감소하였다.
- 수요지당 수거함까지의 평균 거리도 기존 권역은 약 548m, 최적 권역은 약 362m로 186m 단축되어 보행 접근성이 개선되었다.

**폐의약품 수거함이 커버하는 세대 가구 수**
- 최적 입지에서 하나의 수거함이 커버하는 평균 세대 수는 1,689가구로, 기존 권역의 3,378가구보다 절반 이상 감소하였다.
- 기존 권역의 표준편차는 1,662가구였던 반면, 최적 권역은 464가구로 편차가 줄어들어 공간적 효율성이 향상되었다.

## 6. 어려웠던 점과 해결방법 
- 후보지와 수요지간의 도보 거리 행렬을 구축하기 위해 다양한 플랫폼의 API를 사용하여 데이터 수집을 시도하였으나 차량 기준으로만 경로를 구할 수 있다는 어려움이 있었다. 이를 해결하기 위해 T map API를 활용하여 거리 행렬을 구축할 수 있었고 경로 시각화 또한 도보 경로를 기준으로 구현할 수 있었다.
- 후보지 데이터로 수거시설의 개수 및 위치 데이터가 필요했는데 공공데이터로 수집할 수 있는 데이터가 한정적이라는 어려움이 있었다. 이를 해결하기 위해 다양한 포털에 흩어진 데이터들을 수합하는 과정을 거쳤고 위치 데이터와 같은 경우에는 위도, 경도 데이터를 수집하기 위해 지오코딩을 통해 해결할 수 있었다.
